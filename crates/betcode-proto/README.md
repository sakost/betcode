# betcode-proto

Generated protobuf code for the BetCode gRPC API.

## Overview

This crate contains the Rust types generated by `tonic-build` from the protobuf definitions
in `proto/betcode/v1/`. It provides all gRPC service definitions, request/response types,
and shared message types used across BetCode crates.

## Regenerating

After editing `.proto` files in the `proto/` submodule:

```bash
cargo build -p betcode-proto
```
