# betcode-core

Shared types, configuration parsing, and error definitions for BetCode.

## Overview

This crate provides the foundational types shared across all BetCode crates:

- NDJSON message types for the Claude Code subprocess protocol
- Configuration types and resolution logic
- Permission rule parsing
- Shared error types (via `thiserror`)
